¿Qué es el método científico?

El método científico es un proceso organizado por medio del cual se busca generar conocimiento, previamente comprobado y verificado.

Este método consta de un conjunto de pasos determinados, a continuación te los explicaré:

  1. Observación: consiste en ver el entorno que se quiere estudiar, de manera detallada
  2. Preguntas: en esta fase se plantea el problema que se quiere investigar.
  3. Hipótesis: son las teorías que se tienen sobre el problema, las que se van a confirmar o refutar.
  4. Experimentación: son las distintas pruebas que se hacen para intentar comprobar la hipótesis.
  5. Conclusiones: es la presentación de los resultados, en dónde se avala o refuta la hipótesis.
